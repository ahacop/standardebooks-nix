# se unicode-names

```
USAGE

	unicode-names [-h,--help] [<STRING> ...]

DESCRIPTION

	Display Unicode code points, descriptions, and links to more details
	for each character in a string. Useful for differentiating between
	different flavors of spaces, dashes, and invisible characters like
	word joiners.

POSITIONAL ARGUMENTS

	[<STRING> ...]

		A Unicode string.

OPTIONS

	-h,--help

		Show this help message and exit.
```
