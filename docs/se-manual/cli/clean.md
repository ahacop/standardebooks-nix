# se clean

```
USAGE

	clean [-h,--help] [-v,--verbose] <TARGET> [<TARGET> ...]

DESCRIPTION

	Prettify and canonicalize individual XHTML, SVG, or CSS files, or all
	XHTML, SVG, or CSS files in a source directory.

POSITIONAL ARGUMENTS

	<TARGET> [<TARGET> ...]

		An XHTML, SVG, or CSS file, or a directory containing XHTML,
		SVG, or CSS files.

OPTIONS

	-h,--help

		Show this help message and exit.


	-v,--verbose

		Increase output verbosity.
```
