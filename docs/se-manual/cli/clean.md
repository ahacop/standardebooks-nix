# se clean

```
usage: clean [-h] [-v] TARGET [TARGET ...]

Prettify and canonicalize individual XHTML, SVG, or CSS files, or all XHTML,
SVG, or CSS files in a source directory.

positional arguments:
  TARGET         an XHTML, SVG, or CSS file, or a directory containing XHTML,
                 SVG, or CSS files

options:
  -h, --help     show this help message and exit
  -v, --verbose  increase output verbosity
```
