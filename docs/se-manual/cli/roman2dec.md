# se roman2dec

```
USAGE

	roman2dec [-h,--help] [-n,--no-newline] <NUMERAL> [<NUMERAL> ...]

DESCRIPTION

	Convert a Roman numeral to a decimal number.

POSITIONAL ARGUMENTS

	<NUMERAL> [<NUMERAL> ...]

		A Roman numeral.

OPTIONS

	-h,--help

		Show this help message and exit.


	-n,--no-newline

		Don’t end output with a newline.
```
