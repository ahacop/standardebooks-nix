# se build-images

```
USAGE

	build-images [-h,--help] [-g,--no-generate] [-v,--verbose]
	<DIRECTORY> [<DIRECTORY> ...]

DESCRIPTION

	Generate ebook cover and titlepages for Standard Ebooks ebooks, and
	then build ebook covers and titlepages, placing the output in
	DIRECTORY/src/epub/images/.

POSITIONAL ARGUMENTS

	<DIRECTORY> [<DIRECTORY> ...]

		A Standard Ebooks source directory.

OPTIONS

	-h,--help

		Show this help message and exit.


	-g,--no-generate

		Don’t generate new source cover/titlepage SVGs, only build
		existing ones.


	-v,--verbose

		Increase output verbosity.
```
