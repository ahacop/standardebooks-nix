# se dec2roman

```
USAGE

	dec2roman [-h,--help] [-n,--no-newline] [<INTEGER> ...]

DESCRIPTION

	Convert a decimal number to a Roman numeral.

POSITIONAL ARGUMENTS

	[<INTEGER> ...]

		An integer.

OPTIONS

	-h,--help

		Show this help message and exit.


	-n,--no-newline

		Don’t end output with a newline.
```
