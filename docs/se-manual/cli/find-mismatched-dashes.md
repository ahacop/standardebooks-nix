# se find-mismatched-dashes

```
usage: find-mismatched-dashes [-h] TARGET [TARGET ...]

Find words with mismatched dashes in a set of XHTML files. For example,
`extra-physical` in one file and `extraphysical` in another.

positional arguments:
  TARGET      an XHTML file, or a directory containing XHTML files

options:
  -h, --help  show this help message and exit
```
