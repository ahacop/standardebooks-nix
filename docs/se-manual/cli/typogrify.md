# se typogrify

```
usage: typogrify [-h] [-n] [-v] TARGET [TARGET ...]

Apply some scriptable typography rules from the Standard Ebooks typography
manual to XHTML files.

positional arguments:
  TARGET           an XHTML file, or a directory containing XHTML files

options:
  -h, --help       show this help message and exit
  -n, --no-quotes  don’t convert to smart quotes before doing other
                   adjustments
  -v, --verbose    increase output verbosity
```
